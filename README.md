# Python Certification

Ce dépôt présente mon parcours d’apprentissage en Python dans le cadre des formations Cisco / Python Institute :
- Python Essentials 1
- Python Essentials 2

## Objectif
Construire une base solide en programmation Python à travers une progression structurée, allant des fondamentaux jusqu’aux notions intermédiaires.

## Structure des apprentissages

### Python Essentials 1
Python Essentials 1 correspond à la partie débutante du parcours.

Thèmes principaux étudiés :
- concepts fondamentaux de la programmation ;
- syntaxe et sémantique du langage Python ;
- variables, types de données et opérateurs ;
- entrées / sorties ;
- conditions et boucles ;
- listes, tuples, dictionnaires et chaînes de caractères ;
- fonctions ;
- résolution de problèmes simples avec Python.

### Python Essentials 2
Python Essentials 2 correspond à la partie intermédiaire du parcours.

Thèmes principaux étudiés :
- modules et packages ;
- gestion des exceptions ;
- traitement de fichiers ;
- techniques générales de programmation ;
- programmation orientée objet (OOP).
